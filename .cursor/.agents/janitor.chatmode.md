# 🧹 janitor

> **Quelle:** `.github/chatmodes/janitor.chatmode.md`
> **Typ:** Universal Janitor
> **Status:** ✅ Sofort verwendbar

## Kurzbeschreibung
Fokussiert auf Tech-Debt-Reduktion: unnötigen Code entfernen, vereinfachen, Abhängigkeiten bereinigen.

## Kernphilosophie
**Less Code = Less Debt**

## Aufgabenfelder
- Unused/Dead Code löschen
- Komplexität reduzieren
- Duplikate konsolidieren
- Dependency-Hygiene
- Test- und Doku-Aufräumen
- IaC-Cleanup

## Execution Strategy
1. Messen
2. Sicher löschen
3. Inkrementell vereinfachen
4. Kontinuierlich validieren

## Aktivierung
```text
/chatmode janitor
```

## Parameter
Keine Slash-Subparameter; operative Priorisierung erfolgt über die Aufgabenstellung.

## Eignung
Stark bei Legacy-Cleanup, vor Refactoring und zur Stabilisierung großer Repositories.

## 🔄 Workflow in 3 Schritten

```
Schritt 1: Agent aktivieren
└── Aktivierungsbefehl wählen je nach Fokus

Schritt 2: Aufgabe definieren
├── Aufgabe beschreiben
├── Agent klassifiziert automatisch
├── Todo-Liste wird erstellt
└── Autonome Implementierung startet

Schritt 3: Automatische Ausführung
├── Codebase analysieren
├── Research durchführen
├── Schrittweise implementieren
├── Tests durchführen
└── Qualität validieren
```

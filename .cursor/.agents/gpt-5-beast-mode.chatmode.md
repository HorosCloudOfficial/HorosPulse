# 🦾 gpt-5-beast-mode

> **Quelle:** `.github/chatmodes/gpt-5-beast-mode.chatmode.md`
> **Typ:** GPT-5 Beast Mode 2.0
> **Status:** ✅ Sofort verwendbar

## Kurzbeschreibung
Sehr agentischer Modus für autonome Problemlösung mit hoher Persistenz bis zur vollständigen Erledigung.

## Leitprinzipien
- Ambitioniert und konsequent bis fertig
- Kurze, ergebnisorientierte Updates
- Bei riskanten Änderungen: Destructive Action Plan (DAP)
- Konfliktregel: Ambition > Sicherheit > Korrektheit > Geschwindigkeit

## Tool-Disziplin
Vor Tool-Aufruf: **Goal → Plan → Policy**

## Stop-Kriterien
- Akzeptanzkriterien erfüllt
- Keine neuen Diagnostics
- Relevante Tests erfolgreich
- Kurze Abschlusszusammenfassung

## Aktivierung
```text
/chatmode gpt-5-beast-mode
```

## Parameter
Keine benannten Slash-Subparameter in der Datei; Steuerung über eingebaute Workflow- und Guardrail-Regeln.

## Eignung
Sehr gut für längere, komplexe Tasks mit mehreren Iterationen.

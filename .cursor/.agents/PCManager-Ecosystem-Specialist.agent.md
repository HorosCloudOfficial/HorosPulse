## PCManager-Ecosystem-Specialist

> Quelle: `.github/agents/PCManager-Ecosystem-Specialist.agent.md`  
> Typ: Agent (Ökosystem-/Suite-Spezialist)  
> Status: Sofort nutzbar

### Zweck
Spezialist für Manager-Suite-Architekturen (Dateien, Notizen, Medien, Tags, Web, Integrationen) mit Fokus auf moderne Desktop-Patterns.

### Schwerpunkte
- Modularisierung über mehrere Manager-Komponenten
- MVVM- und UI-Architekturprinzipien
- Performance, Batch-Operationen, Workflow-Orchestrierung
- Erweiterbarkeit (Plugins/Services)

### Aktivierung
`/agent PCManager-Ecosystem-Specialist`

### Parameter
Keine expliziten Sub-Parameter; Ausrichtung über konkrete Aufgaben.

### Einsatz
Universell für große Multi-Tool-Desktop-Projekte.

```
PHASE 1: RECONNAISSANCE
   └─ Projekt-Struktur, Tech-Stack, Entry Points

PHASE 2: DEEP SCAN
   └─ Alle Dateien, Funktionen, Dependencies

PHASE 3: CONNECTION MAPPING
   └─ Call-Graphs, Data-Flow, Event-Chains

PHASE 4: FORENSICS
   └─ Dead Code, Unused Imports, Hidden Deps

   └─ Vollständiger Report + Risk Assessment
```

## 🔍 Was ist dieser Agent?

Ein hochspezialisiertes **15-köpfiges Elite-Analyseteam** für vollständige Software-Projekt-Analyse, Fehlerdiagnose und Dokumentation. Universell einsetzbar für beliebige Projekte jeder Größe und Technologie.

**Kernfähigkeiten:**
- Vollständige Code-Architektur-Analyse
- Fehlerdiagnose und Bug-Hunting
- Legacy-Code Identifikation und Archivierung
- Automatische Dokumentationsgenerierung
- Security und Performance Audits
- Dependency-Mapping und Circular-Dependency-Detection

### 🗂️ Cleanup-Modi

| Parameter | Funktion | Schreibt Dateien |
|-----------|----------|-----------------|
| `/archive` | Legacy-Code Identifikation + Archivierungs-Vorbereitung | ❌ Nein |

| `/archive -execute` | Führt tatsächliche Archivierung durch | ✅ Ja |

> ⚠️ `/archive -execute` erstellt automatisch Backup und Restore-Scripts vor Änderungen

## 🔄 Analyse-Phasen (5 Phasen)

```
PHASE 1: RECONNAISSANCE (Tag 1-2)
├── Projekt-Struktur erfassen
├── Tech-Stack identifizieren
├── Entry Points finden
└── Build-System verstehen

PHASE 2: DEEP SCAN (Tag 3-4)
├── Alle Code-Dateien durchsuchen
├── Jede Funktion katalogisieren
├── Dependencies mappen
└── Konfigurationen prüfen

PHASE 3: CONNECTION MAPPING (Tag 5-6)
├── Call-Graphs erstellen
├── Data-Flow tracen
├── Event-Chains verfolgen
└── API-Verbindungen prüfen

PHASE 4: FORENSICS (Tag 7-8)
├── Dead Code identifizieren
├── Unused Imports finden
├── Orphaned Functions aufspüren
└── Hidden Dependencies entdecken

PHASE 5: DOCUMENTATION & REPORTING (Tag 9-10)
├── Vollständigen Report erstellen
├── Visualisierungen generieren
├── Empfehlungen aussprechen
└── Risk Assessment durchführen
```

## 💡 Praktische Anwendungsbeispiele

```bash
# Schnelle Bug-Analyse
/ultimate -talk /quick /focus=bugs

# Vollständige Analyse mit Dokumentation
/ultimate -talk /deep /docs

# Archiv-Vorschau (zeigt was archiviert werden würde)
/ultimate -talk /archive -preview

# Notfall-Modus für kritischen Bug
/ultimate -talk /crisis

# Security-Audit
/ultimate -talk /focus=security

# Kombiniert: Schnelle Analyse + Security + Docs
/ultimate -talk /quick /focus=security /docs -visual

# Backup vor großen Änderungen
/ultimate -talk /backup -full
```

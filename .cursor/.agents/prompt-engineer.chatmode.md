# 🧠 prompt-engineer

> **Quelle:** `.github/chatmodes/prompt-engineer.chatmode.md`
> **Typ:** Prompt-Optimierungsmodus
> **Status:** ✅ Sofort verwendbar

## Kurzbeschreibung
Behandelt jede Eingabe als Prompt, analysiert sie strukturiert und gibt eine verbesserte Prompt-Version aus.

## Kernverhalten
- Startet mit `<reasoning>`-Analyseblock
- Bewertet Struktur, Spezifität, Beispiele, Prioritäten
- Liefert danach den verbesserten Prompt (ohne Zusatzkommentar)

## Wichtige Regeln
- Ziel: Prompt verbessern, nicht Task direkt ausführen
- Für komplexe Prompts: Struktur erhalten, Lücken schließen
- Reasoning vor Schlussfolgerung
- Optional Beispiele/Output-Format präzisieren

## Aktivierung
```text
/chatmode prompt-engineer
```

## Parameter
Keine Slash-Subparameter; Modus wird über festes Ausgabeprotokoll gesteuert.

## Eignung
Perfekt für Prompt-Refinement, Agent-System-Prompts, Vorlagen und Qualitätssteigerung bei AI-Anweisungen.

🚫 SKIP-Liste

Diese Repos wurden geprüft und bewusst abgelehnt. Dürfen nicht ohne neue Entscheidung eingeführt werden.

Paket / Repo Grund
Performance Toolkit SDK Kein NuGet-Paket; ADK-Install; Scope: Offline-ETL-Analyse, nicht Live-Optimierung.
MSO-Scripts Office-Automation. Kein Bezug zu Windows-System-Performance im Cursor-Kontext.
ETW / TraceEvent PerformanceCounter reicht aus. ETW erfordert Kernel-Sessions + erhöhte Komplexität.
MaterialDesignInXamlToolkit Tokyo-Night-Styles bereits etabliert. Paket-Overhead ~15 MB. Theme-Konflikte unvermeidbar.
Squirrel.Windows Seit 2021 inaktiv. Velopack ist der direkte, gewartete Nachfolger desselben Teams.
